# Hardhik V — AI Engineer Portfolio

## Overview
This is a personal portfolio website for Hardhik Vivekanandha, an AI Engineer, Developer, and Innovator. The site showcases skills, projects, achievements, education, and contact information in a modern, interactive, and visually appealing format.

## Features
- Animated matrix background and custom cursor for a futuristic look
- Responsive design for desktop and mobile
- Navigation bar for quick access to all sections
- Animated transitions and GSAP-powered effects
- Skills, projects, achievements, and education timelines
- Contact form and direct links for email, phone, and social profiles
- Downloadable resume button
- Visitor counter using browser fingerprinting

## Technologies Used
- HTML5, CSS3 (custom properties, grid, flexbox)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Google Fonts (Orbitron, Rajdhani)

## Site Structure
- **Home**: Introduction, tagline, animated interests, and quick contact buttons
- **About**: Personal details, education, and biography (can be expanded)
- **Skills**: Tech stack, proficiency bars, and languages spoken
- **Projects**: Project cards with descriptions, features, and links
- **Achievements**: Timeline of experience and education cards
- **Contact**: Email, phone, SMS, address, social links, and a contact form
- **Navigation Bar**: Fixed at the bottom for easy section switching

## Sections

## Customization
- Update personal details, skills, projects, and achievements directly in the HTML
- Add new projects or achievements by duplicating card elements
- Change theme colors in the CSS `:root` variables
- Replace resume link and contact details as needed

## How to Use
1. Open `portfolio.html` in any modern web browser.
2. Navigate using the bottom navigation bar.
3. Click buttons to contact, download resume, or visit social profiles.
4. For mobile, the layout adapts for optimal viewing.
